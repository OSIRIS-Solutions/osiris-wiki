# Welcome to the OSIRIS Wiki

!!! note "This documentation is constantly being expanded"

    This documentation is continuously being expanded. We welcome contributions and suggestions from the community. If you have any questions about the documentation or if you think something is missing, please contact us via [GitHub](https://github.com/OSIRIS-Solutions/osiris-wiki).

=== "Open"

    OSIRIS is open source and freely available software – completely free of charge.

=== "Smart"

    Collect, analyse, report – OSIRIS takes as much work off your hands as possible.

=== "Intuitive"

    OSIRIS was designed for users: it is simple and intuitive to use.

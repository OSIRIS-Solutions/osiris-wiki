# Who uses OSIRIS?

Here you will find a list of institutes that already use OSIRIS or are in the process of introducing it. This list is not exhaustive, as we do not know about all users and are not permitted to list institutions without their consent, but it should give you an overview of the diversity of the OSIRIS community.

!!! tip "Get in touch!"
    
    OSIRIS is open source software and as such it thrives on the community exchanging ideas and supporting each other. If you use OSIRIS or are in the process of introducing it, then get in touch with us! We look forward to every exchange and every feedback. You can also suggest changes directly via Github and add your institute to this page.

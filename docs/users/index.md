# Anleitungen für Nutzende

OSIRIS hat das Ziel, dir deine Arbeit in deiner individuellen Rolle, sei es die Dokumentation deiner Forschung als Wissenschaftler:in, die Verwaltung des Kollegiums als Personalabteilung oder das Erstellen von Berichten für das Controlling, zu erleichtern. Dafür kann das System durch die Administration passend für eure Bedürfnisse angepasst werden und bietet jeder Rolle unterschiedliche Möglichkeiten.  
In den folgenden Abschnitten werden Funktionen in OSIRIS erklärt, die vielleicht nicht für deine Rolle freigegeben ist. Dies kann individuell bestimmt werden. Kontaktiert eure Administration, wenn euch Funktionen fehlen.
